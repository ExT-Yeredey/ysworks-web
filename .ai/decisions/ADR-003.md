# ADR-003: Use Engineering-First Branding

## Status

Accepted

## Context

YSWORKS is an engineering studio. The public website should communicate quality, restraint, and technical credibility.

The brand should not depend on hype, fake proof, exaggerated AI language, or visual spectacle.

## Decision

Use engineering-first branding for the public website.

The public presentation should prioritize:

- Clarity.
- Performance.
- Accessibility.
- Security awareness.
- Maintainability.
- Calm visual direction.
- Factual writing.

## Consequences

The site should feel modern and premium, but not flashy.

Design and copy should avoid:

- Fake customer logos.
- Invented metrics.
- Generic stock imagery.
- Unsupported claims.
- AI hype.
- Decorative complexity.

Trust should come from execution quality and public clarity.

## Alternatives Considered

- Marketing-first branding: rejected because it increases the risk of unsupported claims and generic presentation.
- Product-dashboard aesthetic: rejected because it could imply public access to private systems.
- AI-hype positioning: rejected because it shifts attention from outcomes to private implementation details.
