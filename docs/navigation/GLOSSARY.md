# YSWORKS Glossary

## Purpose

This glossary routes readers to established meanings. Where a detailed contract
exists, that contract remains authoritative.

## Usage

Use canonical capitalisation when naming an entity from the Canonical Domain
Model. Use ordinary lower case when discussing the general concept rather than
the governed entity.
