# YSWORKS Acronym And Abbreviation Index

## Purpose

This index expands abbreviations used across the repository. It does not make a
technology, provider, standard, or service mandatory.

| Abbreviation | Meaning | Repository context |
| --- | --- | --- |
| ADR | Architecture Decision Record | Accepted technical decision with explicit scope |
| AI | Artificial Intelligence | Capability that augments engineering without replacing responsibility |
| API | Application Programming Interface | Future interface style remains an open implementation decision |
| CDM | Canonical Domain Model | Shared entity and lifecycle contract |
| CI | Continuous Integration | Automated repository validation |
| CTA | Call To Action | Public Website navigation or conversion action |
| DNS | Domain Name System | Public hostname resolution; governed by the public-platform foundation |
| EA | Enterprise Architecture | Citation prefix and enterprise foundation |
| FAQ | Frequently Asked Questions | Public content and interaction pattern |
| MFA | Multi-Factor Authentication | Required assurance for specified internal roles |
| MPA | Multi-Page Application | Public Website navigation model where referenced |
| PR | Pull Request | Reviewable repository change |
| RPO | Recovery Point Objective | Open production recovery decision |
| RTO | Recovery Time Objective | Open production recovery decision |
| SEO | Search Engine Optimisation | Public Website discoverability practice |
| SLA | Service-Level Agreement | Not defined until explicitly approved and agreed |
| TLS | Transport Layer Security | Encrypted network transport |
| TTL | Time To Live | Cache, DNS, or short-lived authorisation lifetime according to context |
| UX | User Experience | Governed experience quality and interaction design |
| WAF | Web Application Firewall | Public edge security control |
| WCAG | Web Content Accessibility Guidelines | Accessibility conformance reference |
| YS | Constitutional YSWORKS monogram | Brand identifier; not expanded as a separate company name |

`n8n` and `Anime.js` are product or library names, not acronyms.
