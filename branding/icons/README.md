# Identity Icons

This folder is reserved for approved favicon, application-icon, and compact
monogram masters derived from the official **YS** logo geometry governed by the
[Brand Bible](../../docs/BRAND_BIBLE.md) (*II.VIII*).

Icons require optical testing at target sizes and in approved dark, light, and
single-colour contexts. They must not introduce a separate product identity or
use an unapproved raster reference.
