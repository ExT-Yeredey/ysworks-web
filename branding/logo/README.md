# Logo Masters

This folder is reserved for Founder-approved editable vector masters and
official lock-ups of the geometric YW identity.

Before adding an asset, complete the governed construction specification in
[Visual Identity](../../docs/company/visual-identity.md), test small and large
use, and record version and approval.

No generated raster, automatic trace, improvised SVG, or production logo is
approved by the existence of this folder.
